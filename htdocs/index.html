Ælde Ænglisc Cweð

This website lets you hear what Old English sounds like when read aloud. Or, rather, what it sounds like when read aloud by computer. It's a work in progress: the quality should improve over time as I learn more about Old English orthography and pronunciation. Simply enter the text to be spoken and click the 'Cweðan' button.

Tips and tricks

You can enter text either as an Old English scholar would (complete with æ's, Þ's and proper macrons such as ē) or you can use a normalised form suitable for people without their keyboards set up for Anglo-Saxon. Use 'ae' for an æsc, 'th' for a þorn and add a 'q' to the end of a vowel in place of a macron, e.g. 'eq' for 'ē' and 'aeq' for 'ǣ'.

Technical note

This site requires a modern browser with support for the <audio> tag and the MP3 codec (so no Firefox for the moment, sorry). The backend is powered by Python and espeak with a custom Anglo-Saxon rule-set which I'm developing. The source-code is available on github.
